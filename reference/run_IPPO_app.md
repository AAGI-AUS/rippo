# Run the Shiny IPPO app

Runs a Shiny app where a user can build, edit and download an IPPO
registry.

## Usage

``` r
run_IPPO_app()
```

## Value

An invisible `NULL`, called for its side-effects of launching a Shiny
application.

## See also

Other create IPPO:
[`create_table_3()`](https://aagi-aus.github.io/rippo/reference/create_table_3.md)
